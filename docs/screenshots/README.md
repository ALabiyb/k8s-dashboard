# Screenshots

Place PNG screenshots here and reference them from the main `README.md`.

Suggested set:

| Filename | Content |
|---|---|
| `overview.png` | Home page — all product tiles with health states |
| `namespace.png` | Namespace drill-down showing individual services |
| `logs.png` | Live pod log viewer with a real stream |
| `tv.png` | TV kiosk view (`/tv` endpoint) — public read-only |
| `email-html.png` | Screenshot of an HTML alert email in a mail client |
| `login-oidc.png` | Login page with SoftNet AD / OIDC button |
| `restart-action.png` | Restart confirmation dialog with success toast |

**Tips for good screenshots:**
- Use a browser at 1280×800 or 1440×900 for readable text
- Prefer real workloads over mock mode if possible — mock has a yellow banner
- Crop to remove OS chrome unless it's relevant
- Compress with `pngquant` or similar before committing (keeps repo small)
